# Hand Gesture Recognition Web Project

Энэхүү төсөл нь **MediaPipe Hands** ашиглан веб камерын тусламжтайгаар гарын хөдөлгөөн, gesture таньж, түүнийгээ веб интерфэйс болон жижиг тоглоомуудтай холбосон туршилтын систем юм.

Төслийн гол санаа нь хэрэглэгч гараа камерын өмнө хөдөлгөхөд веб дээрх объектуудыг удирдах, gesture ашиглан UI action хийх, мөн гар танилтаар тоглоом тоглох боломжийг харуулахад оршино.

---

## Төслийн зорилго

- Веб камер ашиглан гарын байрлал илрүүлэх
- MediaPipe Hands ашиглан real-time hand tracking хийх
- JavaScript дээр gesture recognition logic боловсруулах
- Гарны хөдөлгөөнөөр тоглоом удирдах
- Browser дээр шууд ажиллах interactive demo бүтээх
- Ухаалаг гэрийн simulation болон жижиг game demo-г нэгтгэх

---

## Ашигласан технологи

- **HTML5** – Вебийн үндсэн бүтэц
- **CSS3** – Загвар, responsive layout
- **JavaScript** – Gesture logic, game logic, UI interaction
- **MediaPipe Hands** – Гар таних үндсэн library
- **Canvas API** – Тоглоомын дүрслэл болон animation
- **Browser Camera API** – Веб камер ашиглах

---

## Үндсэн боломжууд

### 1. Hand Gesture Recognition

Веб камерын тусламжтайгаар хэрэглэгчийн гарын landmark-уудыг real-time илрүүлнэ. Илэрсэн гарын байрлал дээр үндэслэн pointer position болон gesture state-ийг тооцоолно.

Жишээ gesture-үүд:

- Гараа зүүн/баруун хөдөлгөх
- Нээлттэй гар
- Атгасан гар
- Заах хөдөлгөөн
- UI control хийх хөдөлгөөн

---

### 2. Smart Home Simulation

Гарын gesture ашиглан ухаалаг гэрийн төхөөрөмжүүдийг удирдаж байгаа мэт simulation хийсэн.

Жишээ нь:

- Гэрэл асаах / унтраах
- Төхөөрөмжийн төлөв өөрчлөх
- Gesture ашиглан control хийх

Энэ хэсэг нь гар танилтыг бодит хэрэглээтэй холбож харуулах зорилготой.

---

### 3. Mini Games

Төслийн хүрээнд гарын хөдөлгөөнөөр удирддаг жижиг тоглоомууд нэмсэн.

#### Argal Collector Game

Энэ тоглоом дээр хэрэглэгч сагс барьсан дүрийг гарын хөдөлгөөнөөр зүүн, баруун тийш хөдөлгөж, дээрээс унаж ирэх аргалыг тосож авна.

Онцлог:

- Камераар гарын хөдөлгөөн танина
- Сагсны дүр зүүн/баруун чиглэлээс хамаарч өөр зурагтай харагдана
- Унаж ирэх объектууд `argal_1.png`–`argal_7.png` зургаас random байдлаар сонгогдоно
- Оноо нэмэгдэх системтэй
- Камер ажиллахгүй үед keyboard/mouse fallback ашиглаж болно

Удирдлага:

- Камераар: гараа зүүн/баруун хөдөлгөх
- Keyboard: `A / D` эсвэл `← / →`
- Mouse: canvas дээр хулганаа хөдөлгөх
- Touch: гар утас, tablet дээр хуруугаар хөдөлгөх
